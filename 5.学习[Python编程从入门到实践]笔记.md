## 1.list 删除指定元素
我们知道一般list可以删除指定位置的元素：`list.pop(0)`或`del list[0]`,其实还有一个删除指定元素：
```
list.remove("YuanSu")
```
注意：这里只删除第一个，如果有多个，需要用循环判断删除。

## 2.list 的深拷贝
```py
>>>a =[1,2,3]
>>>b = a[:]                  #对象拷贝，深拷贝
>>>b[0] = 7
>>>b
[7, 2, 3]
>>>a
[1, 2, 3]
```

```py
c = copy.copy(a)            #对象拷贝，浅拷贝
d = copy.deepcopy(a)        #对象拷贝，深拷贝
```
